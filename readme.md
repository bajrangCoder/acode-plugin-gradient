# Usage
Open command palette(available at editor bottom left corner with `icon -> •••`)
And After it, it's using is like making Maggie(Very easy).

# Features
- Css Gradient Generator 🎨
- Tailwind Gradient Generator 🎨
- User-friendly 🥰
- Preview Mode 🤩
- User-friendly Color Picker 💝
